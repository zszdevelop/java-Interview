# 监控进程并重启

## 1. monitor.sh

```sh
#!/bin/sh
# author zsz
# nohup sh monitor.sh 
AppName=BASE_SERVER.jar

while true;
do
PID=`ps -ef |grep java|grep $AppName|grep -v grep|wc -l`
if [ $PID != 0 ];then
    echo "$AppName is running..."
else
    echo "$AppName is not running..."
    sh /faduit/base-project/server/deploy.sh start
fi
sleep 20
done

```
